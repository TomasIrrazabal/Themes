# **Themes**

### Themes es un repositorio para guardar y preservar las distintas customizaciones que fui creando.

## KDE plasma 6
[KDEplasma](https://github.com/TomasIrrazabal/Themes/tree/main/Horizon%20Theme)

Configuración para tema claro y oscuro.

Cada carpeta incluye su Readme.md con instrucciones y un file sh para la instalación automatica. 
El pack incluye:
- HorizonKDE6-Pack:
  - color-schemes: paletas de color para la configuración del tema global (se aplica en la sección "colores").
  - konsole: paleta de colores para configurar konsole.
- HorizonWallpaper: contiene las 2 imagenes para ambos temas, y sh para incluirlos en el sistema, incluyendolos como wallpaper dinamico.
- Icons: en este caso no son de mi creación, simplemente guardé el archivo .tar para mantener el pack reunido.
- PlasmaStyles: La configuración para cada tema para las aplicaciónes
